Темы в Prizm

Что такое темы и как они работают

Тема — это основа, набор правил в CSS, который определяет, как выглядит проект, построенный с помощью дизайн-системы. Она определяет не только цвета, но ещё шрифты, размер текста, отступы, скругления и тени. Можно использовать несколько тем в одном проекте: «дневную» и «ночную» для всего интерфейса или разные темы для отдельных блоков.

В Prizm доступно две базовые темы: light по умолчанию и 'ночная' - dark. Для кастомизации отдельных стилевых значений, там где это допускается дизайн-системой созданы специальные переменные. Переопределять токены из палитры не рекомендуется.



Как подключить тему

Для того, чтобы подключить тему к проекту и управлять ей, используется атрибут data-prizm-theme. Его необходимо применить на контейнер, тему которому мы устанавливаем, для всего приложения в качестве контейнера рекомендуется использовать body


В проект темы Prizm подключаются с помощью PrizmThemeModule


Так же для использования тем необходимо подключить стили к вашему проекту: либо через angular.json, либо через основной стилевой файл проекта.



Переключение тем

Для переключения тем используется PrizmThemeService. Доступно переключение тем как глобально, так и на отдельный блок приложения.

Так же доступна возможность подключать реверсивные темы с использованием PrizmThemeInvertedValuesService.

Как создать свою тему

Вы можете подключить свою тему в Prizm. Для этого необходимо добавить в проект файлы с переопределенными токенами и палитрами.
