## Zyfra - Breadcrumb

<Subtitle>Breadcrumb provides contextual information about page hierarchy</Subtitle>


Import module:

```typescript
import { ZyfraBreadcrumbModule } from '@ui-platform/components';

@NgModule({
  imports: [
    ZyfraBreadcrumbModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-breadcrumb
  [model]="items"
  [home]="home"
  [style]="style"
  [styleClass]="styleClass"
  (onItemClick)="onItemClickHandler($event)">
</zyfra-breadcrumb>
```

```typescript
public items = [
  { label:'Начало' },
  { label:'Название цеха1' },
```
