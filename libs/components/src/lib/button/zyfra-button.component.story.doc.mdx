## Button

<Subtitle>Button is an extension to standard button element with icons and theming.</Subtitle>

```typescript
npm i @zyfra/zyfra-button
```

Import module:

```typescript
import { ZyfraButtonModule } from '@zyfra/zyfra-button';

@NgModule({
  imports: [ZyfraButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-button label="example button" type="submit" (click)="onClickButton($event)"></zyfra-button>
```

We use our standard css classes for styling :

```
.p-button {
  &-mini
  &-default
  &-success
  &-secondary
  &-info
  &-danger
  &-warning
}
```
