## Zyfra - ConfirmDialog

<Subtitle>
  ConfirmDialog is backed by a service utilizing Observables to display confirmation windows easily that can
  be shared by multiple actions on the same component.
</Subtitle>

Import module:

```typescript
import { ZyfraConfirmDialogModule } from '@ui-platform/components';
import { ZyfraButtonModule } from '@ui-platform/components';

@NgModule({
  imports: [ZyfraConfirmDialogModule, ZyfraButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-button (click)="confirmDialog.onClick($event)" label="for dialog" icon="pi pi-check"></zyfra-button>

<zyfra-confirm-dialog
  #confirmDialog
  message="Вы действительно этого хотите?"
  icon="pi pi-exclamation-triangle"
  header="Confirmation"
  acceptLabel="Удалить"
  rejectLabel="Отменить"
  acceptButtonStyleClass="p-button-danger"
  rejectButtonStyleClass=""
  acceptMessages="[{ severity: 'info1', summary: 'Confirmed1', detail: 'You have accepted1' }]"
  [accept]="acceptForDialog"
  [reject]="rejectForDialog"
></zyfra-confirm-dialog>
```
