## Zyfra - Dialog

<Subtitle>Dialog is a container to display content in an overlay window</Subtitle>

Import module:

```typescript
import { ZyfraDialogModule } from '@digital-plant/zyfra-components';
import { ZyfraButtonModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraDialogModule, ZyfraButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-button
  (click)="visible = !visible"
  label="for dialog"
  icon="zyfra-icon selection-check-simple"
></zyfra-button>

<zyfra-dialog
  [(visible)]="visible"
  [header]="header"
  [position]="position"
  (onShow)="onShow($event)"
  (onHide)="onHide($event)"
  (onResizeInit)="onResizeInit($event)"
  (onResizeEnd)="onResizeEnd($event)"
  (onDragEnd)="onDragEnd($event)"
  (onMaximize)="onMaximize($event)"
>
  Your content
</zyfra-dialog>
```
