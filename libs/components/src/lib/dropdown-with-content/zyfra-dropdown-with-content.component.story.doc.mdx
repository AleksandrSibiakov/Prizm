## Zyfra - DropdownWithContent

<Subtitle>Dropdown is used to select an item from a list of options</Subtitle>

Import module:
```typescript
import { ZyfraDropdownWithContentModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [
    ZyfraDropdownWithContentModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-dropdown-with-content
  [options]="cities"
  [(ngModel)]="model"
  optionLabel="name"
  placeholder="Select a City"
  [showClear]="true"
  [mini]="true"
>
  <your-left-projected-content left></your-left-projected-content>
  <your-right-projected-content right></your-right-projected-content>
</zyfra-dropdown-with-content>
```

```typescript
public cities = [
  { name: 'New York', code: 'NY' },
  { name: 'Rome', code: 'RM' },
  { name: 'London', code: 'LDN' },
  { name: 'Istanbul', code: 'IST' },
  { name: 'Paris', code: 'PRS' }
];
public model = null;
```
