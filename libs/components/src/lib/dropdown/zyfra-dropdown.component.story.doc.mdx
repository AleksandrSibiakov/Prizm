## Zyfra - Dropdown

<Subtitle>Dropdown is used to select an item from a list of options</Subtitle>

```
npm i @zyfra/zyfra-dropdown
```

Import module:
```typescript
import { ZyfraDropdownModule } from '@zyfra/zyfra-dropdown';

@NgModule({
  imports: [
    ZyfraDropdownModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-dropdown
  [options]="citites"
  [(model)]="model"
  optionLabel="name"
  placeholder="Select a City"
  [showClear]="true"
></zyfra-dropdown>
```

```typescript
public citites = [
  { name: 'New York', code: 'NY' },
  { name: 'Rome', code: 'RM' },
  { name: 'London', code: 'LDN' },
  { name: 'Istanbul', code: 'IST' },
  { name: 'Paris', code: 'PRS' }
];
public model = null;
```
