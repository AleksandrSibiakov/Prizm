## Zyfra - Hint

Import module:

```typescript
import { ZyfraHintModule, ZyfraHintOverlayManager } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraHintModule],
  provides: [ZyfraHintOverlayManager],
})
export class YourModule {}
```

