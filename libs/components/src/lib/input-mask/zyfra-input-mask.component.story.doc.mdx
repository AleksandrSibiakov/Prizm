## Zyfra - InputMask

<Subtitle>InputMask component is used to enter input in a certain format such as numeric, date, currency and phone</Subtitle>

Import module:
```typescript
import { ZyfraInputMaskModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [
    ZyfraInputMaskModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-input-mask
  [(model)]="modelForInputMask"
  mask="99-9999"
  slotChar=" "
  placeholder="99-999999"
></zyfra-input-mask>
```

```typescript
public modelForInputMask = '';
```
