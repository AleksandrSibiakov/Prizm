## Zyfra - InputNumber

<Subtitle>InputNumber is an input component to provide numerical input</Subtitle>

```
npm i @zyfra/zyfra-input-number
```

Import module:

```typescript
import { ZyfraInputNumberModule } from '@zyfra/zyfra-input-number';

@NgModule({
  imports: [ZyfraInputNumberModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-input-number
  [(model)]="modelForInputNumber"
  showButtons="true"
  mode="currency"
  currency="USD"
></zyfra-input-number>
```

```typescript
public modelForInputNumber = '';
```
