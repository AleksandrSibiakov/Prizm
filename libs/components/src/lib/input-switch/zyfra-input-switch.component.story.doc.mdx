## Zyfra - InputSwitch

<Subtitle>InputSwitch is used to select a boolean value</Subtitle>

```
npm i @zyfra/zyfra-input-switch
```

Import module:

```typescript
import { ZyfraInputSwitchModule } from '@zyfra/zyfra-input-switch';

@NgModule({
  imports: [ZyfraInputSwitchModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-input-switch
  [(checked)]="checked"
  (onChange)="onChangeInputSwitchHandler($event)"
></zyfra-input-switch>
```

```typescript
public checked = false;
```
