## Zyfra - Input

<Subtitle>InputText renders a text field to enter data</Subtitle>

```
npm i @zyfra/zyfra-input
```

Import module:

```typescript
import { ZyfraInputModule } from '@zyfra/zyfra-input';

@NgModule({
  imports: [ZyfraInputModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-input
  id="id"
  type="text"
  [(model)]="inputModel"
  placeholder="placeholder"
  inputClass="input-class"
></zyfra-input>
```

```typescript
public inputModel = 'some string';
```
