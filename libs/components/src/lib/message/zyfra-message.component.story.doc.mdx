## Zyfra - Message

<Subtitle>Messages is used to display alerts inline</Subtitle>

```
npm i @zyfra/zyfra-message
```

Import module:

```typescript
import { ZyfraMessageModule } from '@zyfra/zyfra-message';

@NgModule({
  imports: [ZyfraMessageModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-message [value]="messages"></zyfra-message>
```

```typescript
public messages = [
  {
    severity: 'success',
    summary: 'Success',
    detail: 'Message Content',
    life: 1000
  }
];
```

Other example:

```html
<zyfra-message [value]="messages"></zyfra-message>
```

```typescript
public messages = [
  {
    severity: 'success',
    summary: 'Success',
    detail: 'Надпись появляется плавно',
    life: 3000
  },
  {
    severity: 'error',
    summary: 'Error',
    detail: 'Сообщение ошибки',
    life: 2000
  }
];
```
