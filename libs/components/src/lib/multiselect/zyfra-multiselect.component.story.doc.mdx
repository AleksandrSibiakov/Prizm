## Zyfra - MultiSelect

<Subtitle>MultiSelect</Subtitle>

Import module:

```typescript
import { ZyfraMultiSelectModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraMultiSelectModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-multiselect
  [(ngModel)]="model"
  [options]="cities"
  [optionLabel]="name"
  [placeholder]="Select a City"
></zyfra-multiselect>
```

```typescript
public cities = [
  { name: 'New York', code: 'NY' },
  { name: 'Rome', code: 'RM' },
  { name: 'London', code: 'LDN' },
  { name: 'Istanbul', code: 'IST' },
```
