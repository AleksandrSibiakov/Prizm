## Zyfra - Paginator

<Subtitle>Paginator is a generic component to display content in paged format</Subtitle>

```typescript
npm i @zyfra/zyfra-paginator
```

Import module:

```typescript
import { ZyfraPaginatorModule } from '@zyfra/zyfra-paginator';

@NgModule({
  imports: [ZyfraPaginatorModule],
})
export class YourModule {}
```

Component example:

```html
<zyfra-paginator [rows]="10" [totalRecords]="100" [rowPageOptions]="[25, 50, 75, 100]"></zyfra-paginator>
```
