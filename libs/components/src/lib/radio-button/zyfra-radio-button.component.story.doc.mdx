## Zyfra-RadioButton

<Subtitle>RadioButton is an extension to standard radio button element with theming</Subtitle>

Import module:

```typescript
import { ZyfraRadioButtonModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraRadioButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-radio-button
   *ngFor="let item of items"
   [name]="name"
   [styleClass]="styleClass"
   [label]="item.label"
   [value]="item.value"
   [disabled]="disabled"
   [(ngModel)]="model"
   (onClick)="onClick($event)"
   (ngModelChange)="ngModelChange($event)"
   (onFocus)="onFocus($event)"
   (onBlur)="onBlur($event)"
></zyfra-radio-button>
```

```typescript
const items = [
  {
    label: 'red',
    value: 'red',
  },
  {
    label: 'black',
    value: 'black',
  },
  {
    label: 'blue',
    value: 'blue',
  },
  {
    label: 'pink',
    value: 'pink',
  },
  {
    label: 'Не заполнено',
    value: null,
  },
];
```
