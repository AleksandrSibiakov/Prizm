## Zyfra-RadioButton

<Subtitle>RadioButton is an extension to standard radio button element with theming</Subtitle>

Import module:

```typescript
import { ZyfraRadioButtonModule } from '@ui-platform/components';

@NgModule({
  imports: [ZyfraRadioButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-radio-button
  [model]="radioModel"
  [(selectedValue)]="selectedRadioValue"
  (onClick)="onClickRadioButton($event)"
></zyfra-radio-button>
```
