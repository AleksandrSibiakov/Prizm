## Zyfra-SelectButton

<Subtitle>
  SelectButton is a form component to choose a value from a list of options using button elements
</Subtitle>

Import module:

```typescript
import { ZyfraSelectButtonModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraSelectButtonModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-select-button
  [options]="cities"
  optionLabel="name"
  optionDisabled="inactive"
  [multiple]="true"
  optionValue="code"
  styleClass="p-button-info"
></zyfra-select-button>
```

```typescript
interface City {
  name: string;
  code: string;
}

this.cities: City[] = [
  { name: 'Saint-Petersburg', code: 'SPB' },
  { name: 'Rome', code: 'RM' },
  { name: 'London', code: 'LDN' },
  { name: 'Istanbul', code: 'IST' },
  { name: 'Paris', code: 'PRS '}
];
```

We use our standard css classes for styling :

```
.p-selectbutton {
  &-mini
  &-default
  &-success
  &-secondary
  &-info
  &-danger
  &-warning
}
```
