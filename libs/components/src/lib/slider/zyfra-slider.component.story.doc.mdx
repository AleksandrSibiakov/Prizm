## Zyfra - Slider

<Subtitle>Slider is an input component to provide a numerical input</Subtitle>

To module:
```typescript
import { ZyfraSliderModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [
    ZyfraSliderModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-slider
  [(model)]="model"
  min="0"
  max="100"
  step="1"
></zyfra-slider>
```

```typescript
public model = 20;
```
