## Zyfra - TabMenu

<Subtitle>TabMenu is a navigation component that displays items as tab headers</Subtitle>

To module:
```typescript
import { ZyfraTabMenuModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [
    ZyfraTabMenuModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-tab-menu
  [class]="zyfraTabMenuTagStyle"
  [activeIndex]="activeIndex"
  [controlClose]="controlClose"
  [style]="style"
  [styleClass]="styleClass"
  (onChange)="onChangeHandler($event)"
  (onClose)="onCloseHandler($event)"
>
</zyfra-tab-menu>
```
