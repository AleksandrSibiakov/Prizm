## Zyfra - Table

<Description of={ZyfraTableComponent}>Table displays data in tabular format.</Description>


Import module:

```typescript
import { ZyfraTableModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [
    ZyfraTableModule,
  ]
})
export class YourModule {}
```

Component example:

```html
<zyfra-table
  [value]="value"
  title="Базовая таблица"
  responsiveLayout="scroll"
>
  <ng-template zyfraTableTemplate="header">
    <tr>
      <th>Code</th>
      <th>Name</th>
      <th>Category</th>
      <th>Quantity</th>
    </tr>
  </ng-template>
  <ng-template zyfraTableTemplate="body" let-item>
    <tr [zyfraRow]="item">
      <td>{{ item.code }}</td>
      <td>{{ item.name }}</td>
      <td>{{ item.category }}</td>
      <td>{{ item.quantity }}</td>
    </tr>
  </ng-template>
</zyfra-table>
```
