## Zyfra - Textarea

<Subtitle>Textarea is a multi-line text input element</Subtitle>

```
npm i @zyfra/zyfra-textarea
```

Import module:
```typescript
import { ZyfraTextareaModule } from '@zyfra/zyfra-textarea';

@NgModule({
  imports: [
    ZyfraTextareaModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-textarea
  rows="5"
  cols="10"
  autoResize="false"
  [(model)]="modelForTextArea"
  placeholder="Введите текст.."
  class="inputClass"
  disabled="false"
  maxlength="100"
  minlength="0"
  name="some name"
  (change)="changeTextAreaHandler($event)"
  (onResize)="onResizeTextAreaHandler($event)"
></zyfra-textarea>
```

```typescript
public modelForTextArea = '';

onResizeTextAreaHandler(event) {
  console.log(event);
}
```
