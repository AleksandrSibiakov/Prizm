## Zyfra - Textarea

<Subtitle>Textarea is a multi-line text input element</Subtitle>

Import module:

```typescript
import { ZyfraTextareaModule } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraTextareaModule],
})
export class YourModule {}
```

To component:

```html
<zyfra-textarea
  rows="5"
  cols="10"
  autoResize="false"
  [(ngModel)]="modelForTextArea"
  placeholder="Введите текст.."
  class="inputClass"
  disabled="false"
  maxlength="100"
  minlength="0"
  name="some name"
  (onResize)="onResizeTextAreaHandler($event)"
></zyfra-textarea>
```

```typescript
public modelForTextArea = '';
```
