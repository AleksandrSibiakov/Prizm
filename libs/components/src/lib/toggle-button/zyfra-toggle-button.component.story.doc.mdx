## Zyfra-ToggleButton

<Subtitle>ToggleButton is used to select a boolean value using a button</Subtitle>

```
npm i @zyfra/zyfra-toggle-button
```

Import module:
```typescript
import { ZyfraToggleButtonModule } from '@zyfra/zyfra-toggle-button';

@NgModule({
  imports: [
    ZyfraToggleButtonModule,
  ]
})
export class YourModule {}
```

To component:
```html
<zyfra-toggle-button
  onLabel = "onLabel"
  offLabel = "offLabel"
  onIcon = "onLabel"
  offIcon = "offLabel"
  styleClass="p-button-info"
></zyfra-toggle-button>
```

We use our standard css classes for styling :

```
.p-togglebutton {
  &-mini
  &-default
  &-success
  &-secondary
  &-info
  &-danger
  &-warning
}
```
