## Zyfra - Tooltip

<Subtitle of={ZyfraTooltipDirective}>
  Tooltip provides functionality of additional information for an element
</Subtitle>

Import module:

```typescript
import { ZyfraTooltipModule, ZyfraTooltipOverlayManager } from '@digital-plant/zyfra-components';

@NgModule({
  imports: [ZyfraTooltipModule],
  provides: [ZyfraTooltipOverlayManager],
})
export class YourModule {}
```

Tooltip with text:

```html
<button zyfraButton class="class" type="button"
        zyfraTooltip="Подсказка справа"
        zyfraTooltipPosition="right">Right</button>
```


Tooltip with template:
```html
<ng-template #someTemplate><span>Show me</span></div></ng-template>
<button zyfraButton class="class" type="button"
        [zyfraTooltip]="someTemplate"
        color="success"
        zyfraTooltipPosition="right">Right</button>
```

Tooltip with context:
```html
<button zyfraButton class="class" type="button"
        zyfraTooltip="Подсказка слева"
        [zyfraTooltipContext]="{testContext: '...'}"
        color="success"
        zyfraTooltipPosition="left">Left</button>
```
